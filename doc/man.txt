

VARIABLES

all the variables are declared with the keyword "var"

the name of a variable must contain only letters, numbers, "$", "_". The fist caracter can't be a number

if a variable isn't declared with "var" then it's a global variable


DATA TYPES

1 numbers (integer, float)
2 strings (it can be with double or simple quotes and it can contain scape characters)
3 arrays (it can contain one type or many types)
4 booleans (it can contain true or false)



OPERADORES
 asignacion =
 incremento ++
 decremento --
 negacion ! (for obtain the negation from a data type different to a boolean first the type is converted to a boolean type)
 and &&
 or ||
 mathematics +, -, * , /, %
 relacionales >, <, >=, <=, == !=


 javascript has to scope for the variables (global, local)

 a variable defined out of a function becames a global scope
